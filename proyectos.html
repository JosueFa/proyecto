Evasor de obstáculos

Arduino C++ Sensores

Descripción Detallada

El robot evasor de obstáculos es un proyecto de robótica autónoma que utiliza sensores ultrasónicos para detectar y evitar obstáculos en su camino. El robot es capaz de navegar de forma independiente en entornos dinámicos.

Características Principales

  • Detección de obstáculos en tiempo real
  • Navegación autónoma
  • Sistema de control adaptativo

Tecnologías Utilizadas

Arduino UNO, sensores ultrasónicos HC-SR04, motores DC, programación en C++, y diseño de PCB personalizado.

Robot Manejable

Arduino Bluetooth Android

Descripción Detallada

Robot controlado mediante una aplicación móvil que permite el control remoto en tiempo real. La comunicación se realiza mediante Bluetooth, permitiendo un control preciso de los movimientos.

Características Principales

  • Control mediante aplicación móvil
  • Conexión Bluetooth
  • Respuesta en tiempo real

Tecnologías Utilizadas

Arduino, módulo Bluetooth HC-05, Android Studio para la app móvil, motores DC con puente H.

Seguidor de Línea

Arduino Sensores IR PID

Descripción Detallada

Robot seguidor de línea que utiliza sensores infrarrojos para detectar y seguir una línea negra sobre fondo blanco. Implementa un control PID para un seguimiento suave y preciso.

Características Principales

  • Control PID para seguimiento preciso
  • Array de sensores IR
  • Velocidad ajustable

Tecnologías Utilizadas

Arduino, sensores IR QTR-8A, algoritmo PID, motores DC con encoders.
